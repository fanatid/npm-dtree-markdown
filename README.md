## Installation

```shell
$ npm i npm-rdtree-markdown -g
$ npm-rdtree-markdown crypto-browserify
...
$ cd work-directory/crypto-browserify
$ npm-rdtree-markdown
... (will automatically use package.json from current directory)
```
